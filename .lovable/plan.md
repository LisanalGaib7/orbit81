# NOVA & PIXEL 리디자인

## 변경 내역

### 1. NOVA — 소년 느낌으로 리모델링

- 현재 문제: 너무 성숙하고 느끼한 인상 (또렷한 턱선, 어른스러운 눈빛)
- 변경:
  - 어리고 풋풋한 **소년** 인상 (둥근 볼, 큰 눈, 살짝 미소)
  - 헤어: 단정하지만 좀 더 자연스럽고 보송한 짧은 블론드 (옆가르마 유지하되 부드럽게)
  - 표정: 호기심 가득한 신참 파일럿
- 의상/팔레트/포즈/배경: 그대로 (크림+골드 슈트, 헬멧 옆구리, 검정 배경)

### 2. PIXEL — 긴머리 미녀 (카리나 느낌) + 흰 테두리 제거

- 현재 문제:
  - 단발이 너무 짧고 표정이 딱딱함 (무뚝뚝)
  - 캐릭터 외곽에 하얀 스티커 같은 테두리가 둘러짐
- 변경:
  - 헤어: **긴 생머리 검정**, 가르마 정돈된 시크한 스타일
  - 얼굴: 고양이상 미녀 (살짝 올라간 눈매 유지) + **부드러운 표정과 옅은 미소**, 세련되고 도시적인 인상 (에스파 카리나 레퍼런스)
  - **흰색 외곽 테두리 완전 제거** → 캐릭터 외곽선은 기존 도트 아웃라인(다크)만 남기고 배경은 순수 검정 #000000
- 의상/팔레트/포즈: 변경 (실버 미니멀 슈트 + 시안 액센트, 헬멧 바닥) + 한손은 총들고 있는 걸로

## 기술 노트

- `imagegen--edit_image` 사용, 업로드 레퍼런스 도트 스타일 일관성 유지
- 출력 경로 동일: `src/assets/avatars/nova.png`, `src/assets/avatars/pixel.png`
- `index.ts` 및 컴포넌트 코드 변경 없음
- 나머지 6개 아바타는 건드리지 않음

## 작업 범위 외

- 다른 캐릭터 (EMBER/NEBULA/DRIFT/RELIC/AURORA/COMET) 변경 없음
- 카탈로그 메타데이터 (이름/태그라인/액센트 컬러) 변경 없음
