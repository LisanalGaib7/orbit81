# 아바타 4가지 수정

업로드해주신 도트 캐릭터의 스타일/비율/아웃라인은 그대로 유지하면서, 배경만 검정으로 통일하고 3개 캐릭터를 리디자인합니다.

## 변경 내역

### 1. NOVA — 헤어 스타일 변경
- 기존: 노란 스파이키 머리 (앞으로 뻗침)
- 변경: 차분하고 단정한 짧은 사이드 파트 헤어 (여전히 블론드, 좀 더 파일럿스러운 정돈된 느낌)
- 의상/팔레트/포즈: 유지 (크림+골드 슈트, 헬멧 옆구리)

### 2. DRIFT — 여성 캐릭터 + 헬멧 OFF
- 기존: 둥근 헬멧 풀착용, 얼굴 가려짐
- 변경: 헬멧 벗고 손에 들거나 옆에 둠
- 캐릭터: 귀여운 여자 파일럿, 부드러운 단발 또는 트윈테일, 큰 눈, 살짝 미소
- 팔레트: 기존 teal/cyan #4FC3C7 유지
- 슈트: 기존 teal 슈트 그대로

### 3. PIXEL — 고양이상 미인 여캐 + 헬멧 OFF
- 기존: 검은 바이저 헬멧 풀착용
- 변경: 헬멧 벗음
- 캐릭터: 고양이상 미인 (날카롭고 끝이 살짝 올라간 눈매, 차분한 표정, 단정한 짧은 검정 또는 실버 헤어)
- 팔레트: 기존 실버/화이트 #E5E7EB + 시안 액센트 유지
- 슈트: 미니멀 실버 슈트 그대로

### 4. 배경 — 전 캐릭터 검정 통일
- 8개 전부 (NOVA, EMBER, NEBULA, DRIFT, RELIC, PIXEL, AURORA, COMET)를 처음 버전처럼 **순수 검정 배경 (#000000)** 으로 재생성
- 즉 NOVA/DRIFT/PIXEL은 새 디자인 + 검정 배경, 나머지 5개(EMBER/NEBULA/RELIC/AURORA/COMET)는 현재 디자인 유지하되 배경만 검정으로 다시 뽑음

## 기술 노트

- `imagegen--edit_image` 사용, 업로드해주신 레퍼런스 도트 스타일을 기준으로 일관성 유지
- 출력 경로는 기존 그대로 (`src/assets/avatars/{id}.png`) — `index.ts` 코드 변경 없음
- `PilotAvatar` 의 `image-rendering: pixelated` 덕분에 작은 사이즈에서도 도트가 살아남
- 카드/셀렉션 액센트 컬러(`accent`)도 기존 값 유지

## 작업 범위 외

- 카탈로그 ID, 이름, 태그라인, 컴포넌트 로직, 레이아웃은 변경하지 않음
- 캐릭터 추가/삭제 없음
